# iot-mesh-schema
Schematics and PCB for IoT sensor device

## Description

This PCB is used to connect Meshtastic Node with Wemos d1 mini arduino.

Temperature and humidity/pressure sensors are connected to Wemos.

Firmware for Wemos is  [arduino-iot-sensors](https://github.com/korsumaki/arduino-iot-sensors) repository.
